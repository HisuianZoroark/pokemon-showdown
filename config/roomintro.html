      Come on and Mash, and welcome to the Trash

Trash Channel Original Programming:-
Mix and Meta - Pit sets from a variety of popular OMs against each other to see which is the strongest. (Check compatible metas by using /om mixandmeta).
Bitch and Beggar - Pokémon can 'Beggar-Evolve' using low (<=300) BST Pokémon as Stones.
BEA5T M0D3 - Pokémon in this ZU-based meta can set other species as moves (including Ubers and Mega formes) to transform into them.
The Call of Pikacthulhu - Pokémon get Perish status applied when entering battle.

Added tour commands:-
R 350 Cup Rule - Adds 350 Cup rules to battles
Camomons Rule - Adds Camomons rules to battles
Reversed Rule - Adds Reversed rules to battles
Tier Shift Rule - Adds Tier Shift rules to battles
Averagemons Rule - Adds Averagemons rules to battles
Pokebilities Lite Rule - Adds main Pokebilities mechanic to battles
Suicide Cup Standard Package - Suicide Cup win condition and validation

Added chat commands:-

/mnmts - /mnm + /ts
/bnb - See the effects of a Beggar Evolution. E.g. /bnb lycanroc-midnight @ geodude-alola
